SEO

SEO, or search engine optimization, is the practice of making your website easier for search engines to read and interpret, while also helping people discover your site and decide if it’s worth clicking on when it appears in search results. It involves things like using clear page titles, relevant keywords, and well-structured content so both search engines and users can quickly understand what your page is about.


Some of the most important header tags you should use on your pages for SEO include:

  • The title tag under the head tag, conveying the topic what will be found on the page.
  • meta tags such as description and robots help seo understand what is on the page and let crawlers modify their behaviour on the page respectively
  • h1 – The main heading of the page, describing the page’s primary topic and appears once.
  • h2 – Used for major sections under the main heading. Helps organize content into logical parts.
  • h3 - Used to divide further the h2 content.
  • h4 to h6 - Used for more detailed and deeper structure.

    Explanations for various sizings

    1. px is used when exact sizing is required
    2. em is used when sizing should be relative to the parent element, such as nested components that need to scale proportionally
    3. rem (root em) scales relative to the HTML root's font size, giving consistent, predictable sizing
    4. pt is used for print stylesheets and documents meant for printing, as it's based on physical measurements
    5. % is used for proportional adjustments relative to the parent element, intuitive for quick scaling
    6. vw is used when sizing relative to the total width of the visible browser screen

    When to use the various positions

    1. static is used when elements should follow the normal document flow and no special positioning is needed.
    2. relative is used when an element needs small positional adjustments while still keeping its place in the layout.
    3. absolute is used when you want the element to be not in the normal flow, instead in a separate layer; very useful for popup boxes, control menus etc.
    4. fixed is used when an element should stay in the same position on the screen even when the page is scrolled, such as a fixed button.
    5. sticky is used when an element behaves like normal content at first but becomes fixed once a certain scroll position is reached (the footer uses sticky positioning here).
